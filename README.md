# RAG-based Accident Investigation System

## Overview  
The RAG-based Accident Investigation System is designed to enhance the investigation process of accidents by utilizing a Risk Assessment Grid (RAG) framework. This system aims to systematically analyze the causative factors of accidents, provide comprehensive reporting, and improve overall safety measures.

## Features  
- **Risk Assessment:** Identify and evaluate risks associated with different accident scenarios using the RAG methodology.
- **Data Collection:** Gather data from various sources, including eyewitness reports, environmental factors, and previous incident analyses.
- **Reporting Tools:** Generate detailed investigation reports that highlight key findings, contributing factors, and recommendations for future prevention.
- **User-Friendly Interface:** Easy navigation and intuitive design to facilitate seamless use by investigators and safety personnel.

## Objectives  
- To streamline the accident investigation process and ensure thorough evaluations.  
- To minimize the recurrence of similar accidents through effective analysis and reporting.  
- To provide a platform for continuous improvement in workplace safety practices.

## Getting Started  
To get started with the system, follow these steps:
1. Clone the repository:  
   `git clone https://github.com/098ff/acci-RAG-indiv.git`
2. Navigate to the project directory:  
   `cd acci-RAG-indiv`
3. Install the necessary dependencies (if applicable).
4. Launch the application and access the user interface.
